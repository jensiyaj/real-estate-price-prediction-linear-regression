# Real Estate Price Prediction Using Linear Regression

## Goal
To build a linear regression model that predicts house prices based on features such as square footage, number of bedrooms, bathrooms, and location to help buyers and sellers estimate property value.

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Linear Regression

## Key Insights
1. Performed data cleaning and handled missing values in features like area, bedrooms, and location
2. Applied feature engineering and encoding techniques for categorical variables like location and property type
3. Built and trained a Linear Regression model to predict house prices with R² score and RMSE metrics
4. Visualized actual vs predicted prices and feature importance to interpret model performance

## Dataset
Used real estate dataset containing house features like area, bedrooms, bathrooms, location with corresponding sale prices.
